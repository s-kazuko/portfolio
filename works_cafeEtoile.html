制作背景

不思議な空気に包まれた、魔法がかかったようなショップ。そんな空間に心惹かれ、自分自身が「いつか訪れてみたい」と願うカフェを、Webの世界に描き出しました。幻想と憧れを織り交ぜながら、見る人の心にそっと灯りをともすようなサイトを目指しています。

使用技術

  • HTML / CSS / JavaScript
  • canva / CLIP STUDIO
  • UIアニメーション

技術ポイント

  • CSSアニメーションで空間の広がりを演出
  • JavaScriptでインタラクティブな体験を実装
  • メディアクエリ対応にてモバイルタイプはシンプルに

こだわりポイント

柔らかな人工灯に照らされた画像に、CSSで幻想的なアニメーションを重ねることで、静かに魔法がかかるような演出を施しました。メニュー変更にも柔軟に対応できるよう、商品データはJavaScriptで管理し、更新性と拡張性を両立しています。

制作情報

制作期間：3週間
役割：デザイン・実装

サイトの世界観

まるで魔法の国に迷い込んだかのような、幻想的で心ときめく空間を目指しました。画面越しでも「実際に訪れてみたい」と感じられるよう、細部にまでこだわり、感情に寄り添う演出を心がけています。

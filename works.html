制作背景

ヒーリングとVRの感性とテクノロジーを融合させたプロジェクトを作りたい、そんな願いから生まれました。

使用技術

  • HTML / CSS / JavaScript
  • canva / CLIP STUDIO
  • UIアニメーション

技術ポイント

  • CSSアニメーションで空間の広がりを演出
  • JavaScriptでインタラクティブな体験を実装
  • メディアクエリ対応にてモバイルタイプはシンプルに

こだわりポイント

現代的でスタイリッシュな枠組みの中に感情を揺さぶる優しさを演出し、所々にふわっとした動きを見せるアニメーションで遊び心を散りばめました。

制作情報

制作期間：3週間
役割：デザイン・実装

サイトの世界観

ユーザーがサイトを訪れた瞬間、まるで深呼吸をしたような安心感を得られるように、色彩や動きに細心の注意を払いました。
